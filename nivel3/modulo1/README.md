Shopping CArt
===================

- Backbone
- RequireJS
- SPA

Trabajo: Construir un carrito de compras.
Construir una SPA utilizando HTML5/CSS3, RequireJS, Backbone, jQuery, Underscore y Mustache.
La aplicación debe permitir agregar/eliminar ítems al carrito de compras. Los ítems en venta pueden ser cualquiera que se les ocurra (por ejemplo: macetas, alpargatas, gorros de cotillón, por si no se les ocurre alguno)
La SPA debe contener un slide mostrando cada producto, con una imagen + título + descripción + precio + botón para agregar el producto al carrito de compras indicando la cantidad que se está comprando. Si el producto ya existe en el carrito, se debe indicar esto en lugar del botón.
En la barra de navegación superior, debe crearse un acceso al carrito, en donde se listen los productos ingresados, se permita remover alguno de la lista y permita cambiar la cantidad de productos comprados.
